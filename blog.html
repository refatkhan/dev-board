Question-1: What are the different ways to select an element in the DOM?

    We can select elements in DOM in many ways. We can select with id,class,tag,querySelector, querySelectorAll.

  • document.getElementById("id")
  • document.getElementsByClassName("class")
  • document.getElementsByTagName("tag")
  • document.querySelector(".class")
  • document.querySelectorAll("tag")

Question-3: What is event delegation in the DOM?
In Dom, event delegation is a process where we can add a eventListener in a parent element instead using in every child element. Such as you need same task in every child under a tag. Then you can use event delegation.
Question-4: What is event bubbling in the DOM?
Event bubbling is a mechanism where an event triggered on a child element propagates upward through its ancestors in the DOM. It allows parent elements to respond to events triggered by their child elements Such as we click on a button inside a div, the "click" event will first trigger on the button itself, then bubble up to the div, and potentially to even higher level elements in the DOM depending on how event listeners are set up.
Question-5: How do you create, add, and remove elements using JavaScript?
In JavaScript we can create an element , can add a element and we can also delete an element.
For create a element we can use document.createElement("div");.
For add a element we can use document.body.appendChild(div); .
For remove a element we can use div.remove();
